# Vehicles ORM Project

## Použité technologie
- PHP 8+
- Eloquent ORM
- MySQL
- Composer
- Dotenv

## Funkce
- Připojení k databázi vehicles
- Výpis všech modelů s výrobci
- Filtrování podle výrobce a modelu
- Spojení tabulek modelů a vlastníků

## Spuštění
1. `composer install`
2. Nastav `.env`
3. Spusť `php -S localhost:8000` a otevři `index.php`
